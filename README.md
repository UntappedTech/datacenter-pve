# datacenter-pve

| Directory       | Description                                                                |
| --------------- | -------------------------------------------------------------------------- |
| blockman        | Script which performs CRUD actions on designated blocks in the target file |
| blockman_simple | same as above, but with fewer options/less complexity                      |
| caddy           | Script to generate Caddyfiles from a more convenient JSON format           |
| fedora          | Scripts specific to the Fedora operating system                            |
| templates       | template files and .bashrc.d aliases and functions for lxc guests          |
| .               | scripts which don't fit any current category/used on the PVE Host          |
